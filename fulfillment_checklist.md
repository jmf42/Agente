# Fulfillment Checklist

## Before Payment

- Confirm the website URL.
- Confirm the buyer understands this is an audit and readiness package, not a guaranteed ranking service.
- Send payment link or invoice manually.

## After Payment

1. Run the audit script.
2. Review the generated report manually.
3. Visit the homepage and top service pages.
4. Write the top 3-5 priority fixes in plain English.
5. Draft or refine `llms.txt`.
6. Package the report as Markdown or PDF.
7. Offer optional implementation only if the buyer wants it.

## Quality Bar

- Every recommendation must be specific to the buyer's website.
- Avoid unsupported claims about guaranteed AI rankings.
- Make fixes understandable to a non-technical owner.
- Include exact files or pages to change.
- Include proof: URLs checked, status codes, detected schema, missing files.
