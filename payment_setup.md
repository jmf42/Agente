# Payment Setup Instructions

Do not give an AI agent payment account credentials.

## Safe Payment Options

Use one of these human-controlled options:

1. Create a PayPal payment link for `$149`.
2. Send a PayPal invoice for `$149`.
3. Use Stripe Payment Links for `$149`.
4. Use a marketplace order page if selling through a platform.

## What To Share With The Agent

Safe to share:

- Public payment link.
- Public checkout URL.
- Business support email.
- Public refund policy.

Do not share:

- PayPal password.
- Bank account details.
- Card numbers.
- Two-factor codes.
- API secret keys.
- Marketplace login credentials.

## Suggested Payment Terms

Payment before delivery for first-time buyers.

Refund if the final report does not identify at least three specific improvement opportunities.
